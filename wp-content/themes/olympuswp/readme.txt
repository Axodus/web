Olympus is a blazingly fast, lightweight, fully responsive, SEO-ready, and 100% Vanilla JS WordPress Theme that pursues to deliver quality over quantity while implementing the best coding standards, making it a theme ideal for building scalable sites, from blogs to personal portfolios and mid to large business websites, multisite compatible, and WooCommerce ready. It has been carefully crafted to uphold performance and usability in mind. Built to be compatible with well-coded plugins and all major page builders, including Elementor, Beaver Builder, Visual Composer and Brizy, and most importantly, battle-tested.

== Description ==

Olympus is suitable for any type of WordPress website like portfolio, blog, personal, business, or WooCommerce storefront.
Built with speed in mind, no jQuery or heavy scripts loaded to offer you the fastest WordPress theme possible.

Search engines will love your site built with Olympus witch comes with Schema.org code integrated and best SEO practices.
Works with most page builders like Elementor, Beaver Builder, Visual Composer, SiteOrigin, and Divi.

Many great features: # WooCommerce Ready # Fully Responsive # RTL & Translation Ready # Regularly updated

Still searching for a theme to start your website? We're pretty confident that our theme will be perfect for you! Olympus is a fast, fully customizable and WooCommerce ready theme for building any kind of website!

== Installation ==

= From within WordPress =
1. Visit "Appearance > Themes > Add New"
2. Search for "Olympus"
3. Install and activate

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Olympus support most known plugins like WooCommerce.
